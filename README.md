# KeepIt 🎟️

אפליקציה אישית לניהול שוברים, גיפט קארדים וקופונים — כדי ששום שובר לא יפוג נשכח.
כל הנתונים נשמרים **מקומית בדפדפן** (IndexedDB). אין שרת, אין חשבון, אין עלות.

## הרצה מקומית

```bash
npm install
npm run dev
```

ואז לפתוח את הכתובת שמופיעה בטרמינל (בדרך כלל http://localhost:5173).

## סטאק

| שכבה | כלי |
|---|---|
| UI | React 18 + Vite |
| עיצוב | Tailwind CSS v4 |
| נתונים | IndexedDB דרך Dexie + dexie-react-hooks |

## מבנה הפרויקט

```
src/
├── main.jsx              # נקודת כניסה
├── App.jsx               # דשבורד + ניתוב בין מסכים
├── db.js                 # שכבת הנתונים (Dexie)
├── utils/expiry.js       # חישובי תוקף
└── components/
    ├── VoucherCard.jsx   # כרטיס שובר (בסגנון תלוש)
    ├── AddVoucherForm.jsx
    └── EmptyState.jsx
```
